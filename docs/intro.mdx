## Docz 설정하기.

-   [Docz Configuration](https://www.docz.site/docs/project-configuration)

<br />

### doczrc.js 파일 생성

프로젝트 루트경로에 `doczrc.js` 파일을 생성합니다.

```bash
    touch doczrc.js
```

<br />

### Title 설정하기

`doczrc.js` 내 `title` 프로퍼티를 추가합니다.

```javascript
export default {
    title: "타이틀을 설정합니다."
}
```

<br />

### Description 설정하기

`description` 프로퍼티를 추가합니다.

```javascript
export default {
    title: "타이틀을 설정합니다.",
    description: "설명을 추가합니다"
}
```

<br />

### Menu 설정하기

`.mdx` 파일 내 `metadata`로 메뉴를 설정할 수 있지만 설정파일에서 추가해서 사용할 수도 있습니다.

```javascript
export default {
    title: "타이틀을 설정합니다.",
    description: "설명을 추가합니다",
    menu: [
        "Intro",
        {
            name: "Concept",
            menu: ["Component", "Container", "Store", "Module", "Utility"]
        }
    ]
}
```

사용할 때는 아래와 같이 사용합니다.

<br />

**Intro** 페이지

```markdown
---
name: Intro
---
```

<br />

**Concept/Component** 페이지

```markdown
---
menu: Concept
name: Component
---
```

-   1depth 메뉴를 사용할 때는 menu array 내에 있는 `string` 값과 metadata의 name과 매핑해주면 됩니다.
-   2depth 메뉴를 사용할 때는 menu array 내에 있는 `object` 내 name과 menu 를 metadata의 menu, name과 매핑해주면 됩니다.
